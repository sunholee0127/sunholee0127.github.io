Learning Rate

learning rate에 따라 학습 결과가 달라짐.

  • Learning Rate Decay
    • 처음에는 크게 움직이다가 일정 조건이 되면 learning rate를 낮춰서 점점 작게 움직이는 것
    • constant warmup, gradual warmup
  • Cyclic Learning Rate
    • saddle point를 빠져나갈수 있나?
    • saddle point에서는 learning rate를 키워서 빠져나가야 효과적이기 때문에, learning rate를 주기적으로 변경해보자
